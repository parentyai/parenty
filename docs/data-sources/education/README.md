# 教育データ公式ソース台帳（K-12 / 州別）

## 目的
- 教育領域の公式ソースを一元管理し、情報源の混在と誤読を防ぐ
- 公式性・規約・運用更新の判断基準を揃える
- 将来の実装判断を同じ根拠に固定する

## 対象範囲
- K-12（州・学区・学校）
- 学校暦／休校・遅延
- 学区境界／通学区域
- 給食・食事支援
- 公的告知（州・学区の公式発表）

## 公式性の定義（優先順位）
1. 州教育局（State Education Agency）
2. 州政府（State Government）
3. 学区（District）
4. その他の公的機関・公的ポータル

例外の扱い
- 州が学区情報の責任主体でない場合、学区ソースを公式として扱う
- 州が公式一覧を提示している場合、その一覧を一次根拠とする

## 更新運用
- 台帳の更新担当: レビュー担当（人間）
- 更新頻度: 月次
- レビュー観点:
  - 公式性の根拠URLが追跡できるか
  - 規約URLが存在し、内容が確認できるか
  - 取得範囲が対象外に拡張されていないか
  - 期限切れ・移転・終了が明記されているか

## 禁止事項
- 過度クロールや規約外の取得
- 規約違反のスクレイピング
- PII取得・保存
- 認証情報の直書き
- 公式性の根拠がないソースの追加
